What is Grim Fandango Launcher ?:
A replacement launcher program for Grim Fandango.
It has features not present in the original launcher:

  • Can run Grim Fandango in windowed mode.

  • Can make Grim Fandango run from hard drive (play without the cd's) and can automatically copy the required files to the hard drive.

  • Supports all recent versions of Windows (XP, XP64, XP MCE, Vista, Vista64). Windows 98, ME and 2000 are not officially supported but should work correctly.

  • Automatically enables Grim Fandango's developer/debug mode. See the section below for more information on this.

  • Loads one of 7 random images as the main launcher image - for added eye candy.

The program does not need to be installed and can be run from anywhere since it reads the registry values in order to find the location of Grim Fandango. Consequentally, Grim Fandango must be installed for the launcher to work correctly.


How to use it:
Usage should be fairly self-explanatory:

  • Click either Play fullscreen or Play windowed to start the game.

  • Click Debug Keys to bring up a window that displays the debug keys and their functions.
    These were all found/documented by Serge (Jimmi Thøgersen) with minor additions by me. You do not have to close this window to carry on using the launcher, just move it or minimise it, that way you can refer to it if you want to try the debug keys in windowed mode.

  • Click 'Resume last saved game' to enable/disable the option.
    If enabled Grim will automatically load the last saved game when you click one of the play buttons.

  • The Lucasarts readme button displays Lucasarts' CMI readme, while the Launcher readme button displays this readme.

  • The status bar at the bottom of the launcher shows whether the game will be run from cd or hard drive.

Click Options to bring up the options menu.

  • Close launcher on play:
    If enabled, then the launcher closes when one of the play buttons is clicked.

  • Run Grim from hard drive:
    This governs whether the game is run from hard drive or cd. Click the button to switch between the two options.
    If enabled, the launcher will first check that you have the required files on your hard drive, if not, then it prompts you to insert your Grim Fandango cd's and copies the files to hard drive for you (this requires 1.2 GB of disk space).


Developer Mode/Debug Keys:
The Grim Fandango Developer Mode was found and documented by Serge (Jimmi Thøgersen). His list of debug keys are included in this launcher with his permission.

Developer mode allows the game to be run in a window. This can be done in two ways: by clicking 'Play windowed' in the launcher, or by loading the game, going to options > 3D hardware options and choosing 'DIBSection Windowed Display'.

Developer mode also enables the use of debug keys. The list of debug keys and their functions can be seen by pressing the 'Debug Keys' button in the launcher.

Developer mode is automatically activated by the launcher and even if you then play the game without the launcher, developer mode will still be enabled. I have not provided an option to disable developer mode because this disables some features (windowed mode) and complicates the operation of the launcher. If you do want to disable developer mode though, you can do this by editing the registry. Navigate to in HKEY_LOCAL_MACHINE/Software/LucasArts Entertainment Company LLC/Grim Fandango/v1.0/ and delete the "good_times_ key.


Bugs/Limitations:
Windowed mode may not work on some systems. To fix this you can try activating windowed mode in a different way: In the game, go to Options > 3D Hardware Setup and choose "DIBSection Windowed Display".
